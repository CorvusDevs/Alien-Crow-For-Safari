# Alien Crow For Safari

A Safari extension that supercharges the classic **old.reddit.com** interface with 70+ customizable modules — themes, ad blocker, comment power-ups, saved-content manager, keyboard navigation, and a lot more.

📦 **[Get on the App Store](https://apps.apple.com/app/alien-crow-for-safari/id6769216829)**

🌐 **Website:** https://corvusdevs.github.io/Alien-Crow-For-Safari/

🔒 **[Privacy policy](https://corvusdevs.github.io/Alien-Crow-For-Safari/privacy.html)**

## About

Alien Crow is an independent project. It is not affiliated with, endorsed by, or sponsored by Reddit Inc. The "Reddit" name is used here descriptively to identify the platform this extension enhances.

Localized in 33 languages. Built by [Corvus Devs](https://github.com/CorvusDevs).
