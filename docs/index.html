Alien Crow For Safari

A Safari extension that supercharges the classic old.reddit.com interface with 70+ modules.

Get on the App Store

What it does

Alien Crow brings back the classic browsing experience and powers it up. Open any thread, and Alien Crow takes over: 20+ dark and light themes, an ad & annoyance blocker, comment power-ups, a unified saved-content manager, deleted-comment recovery, an inbox preview, enhanced search, command palette, keyboard navigation, and a lot more — all designed for old.reddit.com.

Dark Mode & 20+ ThemesDracula, Nord, Catppuccin, Solarized, Tokyo Night, OLED black, and more
Ad & Annoyance BlockerStrip promoted posts, app nags, login walls, chat widgets, NFT promos
Comment ToolsAuto-collapse AutoModerator, persistent collapse, jump between top-level comments
Saved Content ManagerUnified viewer for bookmarked posts and saved comments with search and tags
View Deleted ContentOne click to view removed comments via Reveddit
Enhanced SearchAdd sort, time, flair, and operator filters; click any flair to search for it
Command PaletteKeyboard-driven navigation across subreddits, settings, and actions
33 LanguagesFully localized UI with autonym dropdown picker

Privacy

Alien Crow runs entirely in your browser. It does not collect, transmit, or share any data. All settings are stored on your device. Read the full privacy policy.
