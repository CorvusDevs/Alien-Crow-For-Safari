Privacy Policy

Alien Crow for Safari, Last updated 2026-05-13

TL;DR. Alien Crow does not collect, transmit, or share any personal data. Every setting and every cached value stays on your device. There is no analytics, no telemetry, no account, and no server.

What is Alien Crow?

Alien Crow is a Safari Web Extension that enhances the Reddit browsing experience with themes, ad-blocking, keyboard navigation, saved-content tools, and over 70 other features. It runs entirely inside your browser; the host app exists only to install the extension into Safari.

What data Alien Crow collects

None that ever leaves your device. Alien Crow stores the following on your device using Safari's local extension storage:

  • Module preferences, which features you have enabled and any options you have customized (e.g. theme choice, keyboard shortcuts).
  • Per-thread state, counts of comments you have seen so the "new comments" badge works (kept up to 30 days, then pruned).
  • User tags & bookmarks, only if you actively use the User Tagger or Post Bookmarks modules.
  • UI state, your selected display language, whether the welcome toast has been shown, whether the review prompt has been dismissed.

All of this lives in Safari's browser.storage.local sandbox. It is never read by Alien Crow's developer, and it is never sent to any server.

Network access

Alien Crow's content scripts only run on reddit.com pages. It does not make additional network requests except in two opt-in cases:

  • View Deleted Content (off by default), when you click "view deleted" on a removed comment, the extension opens a link to reveddit.com, a third-party archive of Reddit comments. Reveddit's privacy practices are governed by their own privacy policy.
  • RSS Feed (off by default), generates RSS URLs that point at Reddit's own public JSON endpoints; no third-party server is involved.

Permissions

Safari grants Alien Crow the following permissions:

  • storage, to save your preferences locally.
  • activeTab, tabs, to open the Reddit page when you click the toolbar icon, and to refresh the page when you change settings.
  • webNavigation, to redirect new Reddit URLs to old.reddit.com when you've opted into "Force Old Reddit".
  • Host access to *.reddit.com, so the extension can modify Reddit pages.

Alien Crow does not request access to other websites. The host-permission allowlist is literally just Reddit.

Third-party services

Alien Crow is open source and does not embed analytics SDKs, crash reporters, advertising libraries, or social-media trackers. No third-party JavaScript is loaded at runtime.

Children's privacy

Alien Crow does not knowingly process data from children under the age of 13. Since Reddit's own terms require users to be at least 13, and Alien Crow only operates on Reddit, this requirement is enforced upstream.

Changes to this policy

If this policy ever changes, the "Last updated" date at the top will reflect the new revision. Material changes will also be mentioned in the extension's release notes.
